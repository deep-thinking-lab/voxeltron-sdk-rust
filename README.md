# voxeltron-sdk-rust

Rust SDK for Voxeltron.

## Status

This repository is part of the April 17, 2026 Voxeltron org-restructure plan.
The SDK is still staged in the main public repo during the transition:

- upstream path: `plugins/sdk/rust/`
- upstream repo: `deep-thinking-llc/voxeltron`

## Planned Contents

- typed Rust client for public Voxeltron APIs
- examples for embedding and integration tooling
- versioned releases independent from the daemon crate cadence
